# Contributing
Please make changes directly to the `README.md` file. If you're looking to make a large scale change, consider starting a [Discussion](https://github.com/mattantonelli/ffxiv-icons/discussions).

## Resources
A good resource is a website or tool you can use to extract/search/view icons. Add your resource in alphabetical order and provide a short description.

## Reference
Use one or two words to describe the icons available at a given offset. When adding a new offset, make sure it is in numerical order.

In your pull request, provide a simple screenshot showing off some of the relevant icons and their IDs. This helps ensure the validity of your change.
